## Project Overview

This project analyzes Spotify track audio features to understand and predict song popularity tiers.

Using structured Spotify track data, we perform:

- Data acquisition from Kaggle
- Data cleaning and preprocessing
- Feature engineering
- Exploratory Data Analysis (EDA)
- Transformation of popularity into classification tiers for future predictive modeling

This Phase 1 submission covers:
- Problem formulation
- Data collection
- Data cleaning and processing
- Exploratory Data Analysis (EDA)
